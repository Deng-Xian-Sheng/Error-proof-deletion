# Linux防误删除_rm命令的最后一道防线

#### 介绍
人非圣贤，孰能无过；Linux rm命令是及其危险的，数据恢复的代价是沉重的。
本仓库旨在降低风险，为所有Linux使用者保驾护航！

#### 安装教程
将代码添加到全局.bashrc文件中（如果不希望全局也可单独添加在特定用户的.bashrc中）
